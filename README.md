# Blog Website

Full-featured blog website with Node.js + Express

## Features

- User authentication (register/login)
- Create, edit, delete blog posts
- Markdown support
- Comments and likes
- Categories and tags
- Dark/light theme
- Search
- Responsive design

## Setup

```bash
npm install
npm start
```

## Tech Stack

- Backend: Node.js + Express
- Auth: JWT + bcrypt
- Storage: Local JSON files
- Frontend: Vanilla HTML/CSS/JS
